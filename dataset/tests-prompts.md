1. Build something for students.
2. Build a dashboard.
3. Public access but login required.
4. Only admins can access but users should also access.
5. Build an application.
6. Build a scalable system.
7. Create analytics without storing data.
8. Anonymous users must have accounts.
9. Premium users should not pay.
10. Admins and users have identical permissions.
